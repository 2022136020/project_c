# 01. 요구사항 정의 (MoSCoW)

> AI Agent: Claude Code | 최초 작성: 2026-04-28 | 마지막 업데이트: 2026-05-12

---

## Must (없으면 앱이 아닌 것들)

### 인증
- 이메일 회원가입 / 로그인
- Google 소셜 로그인
- 자동 로그인 (앱 재실행 시 유지)

### 가상 계좌 관리
- 주식·코인 계좌 등록 (증권사/거래소별)
- 초기 잔액 입력
- 입금 · 출금 기록 (날짜 + 금액)
- 잔액 자동 계산 (초기 잔액 + 입금 - 출금 + 실현 손익)

### 포지션 기록
- 주식 / 코인 구분
- 진입가 · 수량 · 진입일 기입
- 매수 근거 작성
- 목표가 · 손절가 설정
- 포지션 정리 (정리가 · 정리일 입력)
- 수익률 · 실현 손익 자동 계산

### 복기
- 포지션별 복기 메모 작성 · 조회
- 매수 근거와 실제 결과 비교

---

## Should (핵심 가치를 완성하는 것들)

### 시각화
- 포지션별 수익률 막대 차트
- 기간별 손익 추이 라인 차트
- 기간 필터 (1주 / 1달 / 3달 / 6달 / 1년 / 전체)

### 대시보드
- 자산 현황 요약 (계좌 잔액, 누적 손익, 수익률)
- 최근 포지션 목록

---

## Could (여유가 생기면 추가)

- 대시보드 위젯 커스터마이징 (on/off, 순서 변경)
- 목표가 · 손절가 도달 알림
- 다크 / 라이트 모드
- 다국어 지원 (한국어 기본 / 영어)
- 포지션 태그 분류 (장기 / 단기 / 배당 등)
- 거래 히스토리 연도별 아카이브

---

## Won't (이번 버전 제외)

| 항목 | 이유 |
|------|------|
| 소비 내역 관리 | 실계좌 연동 불가, 현실적 대안 없음 — 보류 |
| 증권사 · 거래소 API 연동 | 구현 불가 |
| 실시간 시세 자동 조회 | 외부 API 의존도 과다 |
| 커뮤니티 게시판 | 핵심 가치와 거리 있음 |
| Apple 소셜 로그인 | 우선순위 낮음 |
| 생체인증 | 우선순위 낮음 |
