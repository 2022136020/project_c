## 3. 기능 요구사항

### 3-1. 인증
| 기능 | 설명 |
|------|------|
| 이메일 회원가입/로그인 | 이메일 + 비밀번호 |
| Google 로그인 | Firebase Auth + google_sign_in |
| 게스트 모드 | 로그인 없이 사용, 로컬 저장, 기기 변경 시 데이터 유실 |
| 자동 로그인 | 앱 재실행 시 로그인 유지 |
| 비밀번호 재설정 | 이메일로 재설정 링크 발송 |

### 3-2. 계좌 관리
| 기능 | 설명 |
|------|------|
| 계좌 등록 | 주식(증권사별) / 코인(거래소별) 구분 등록 |
| 초기 잔액 기입 | 계좌 등록 시 초기 잔액 입력 |
| 입금 기록 | 날짜 + 금액 |
| 출금 기록 | 날짜 + 금액 |
| 잔액 자동 계산 | 초기 잔액 + 입금 - 출금 + 실현 손익 |

### 3-3. 포지션 기록
| 기능 | 설명 |
|------|------|
| 포지션 생성 | 주식/코인 구분, 종목명, 진입 시기, 진입가, 수량, 진입 근거, 목표가, 손절가 |
| 포지션 정리 | 정리 시기, 정리가 입력 |
| 수익률 자동 계산 | (정리가 - 진입가) / 진입가 × 100 |
| 실현 손익 자동 계산 | (정리가 - 진입가) × 수량 |

### 3-4. 시각화
| 기능 | 설명 |
|------|------|
| 포지션별 수익률 그래프 | 종목별 수익률 막대 차트 |
| 실현 손익 추이 그래프 | 기간별 누적 실현 손익 라인 차트 |
| 자산 변동 그래프 | 계좌별 잔액 변동 라인 차트 |
| 기간 필터 | 1주 / 1달 / 3달 / 6달 / 1년 / 전체 / 연도별 |
| 탭 구분 | 종합 / 주식 / 코인 / 종목별 |

### 3-5. 대시보드
| 기능 | 설명 |
|------|------|
| 위젯 on/off | 원하는 위젯만 표시 |
| 위젯 순서 변경 | 드래그로 순서 조정 |
| 설정값 저장 | 로컬에 저장, 앱 재실행 시 유지 |

### 3-6. 개인 설정
| 기능 | 설명 |
|------|------|
| 기준 통화 | KRW / USD |
| 시세 갱신 주기 | 30초 / 1분 / 5분 |
| 알림 설정 | 목표가 도달 알림 on/off |
| 화면 모드 | 다크 / 라이트 |
| 언어 설정 | 한국어 / 영어 / 중국어 |
| 계정 관리 | 닉네임 변경, 비밀번호 변경, 로그아웃, 회원 탈퇴 |

### 3-7. 게시판
| 기능 | 설명 |
|------|------|
| 게시물 작성 | 재테크 정보·투자 전략 게시 |
| 게시물 조회 | 타인 게시물 목록 및 상세 조회 |
| 댓글 | 게시물에 댓글 작성·조회 |
| 좋아요 | 게시물 좋아요 |
| 신고 | 부적절한 게시물·댓글 신고 |

### 3-8. 히스토리
| 기능 | 설명 |
|------|------|
| 거래 내역 | 전체 포지션 진입·정리 내역 |
| 연도별 아카이브 | 연도별 거래 내역 분리 조회 |
