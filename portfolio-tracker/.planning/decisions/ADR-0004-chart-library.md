# ADR-0004: 차트 라이브러리 선택

- 상태: Accepted
- 날짜: 2026-05-19
- 결정자: Jooseonghyeon

## 배경

포지션별 수익률 막대 차트와 누적 손익 라인 차트를 구현하기 위해
Flutter용 차트 라이브러리를 선택해야 한다.

## 고려한 대안

### 대안 A: fl_chart (선택)
- 장점: 오픈소스(무료), Flutter 전용 설계, BarChart/LineChart/PieChart 모두 지원, pub.dev 2,100+ 좋아요, 커스터마이징 자유도 높음
- 단점: 복잡한 인터랙션 구현 시 코드가 장황해질 수 있음

### 대안 B: syncfusion_flutter_charts
- 장점: 엔터프라이즈급 기능, UI 완성도 높음
- 단점: 상업적 사용 시 라이선스 필요, 무료 티어 워터마크 존재

### 대안 C: charts_flutter (Google)
- 장점: Google 제작
- 단점: 2022년 이후 업데이트 없음, 사실상 지원 중단

## 결정

**fl_chart `^0.70.2`**를 사용한다.

## 이유

- 완전 무료·오픈소스로 라이선스 제약 없음
- 이 프로젝트에 필요한 BarChart, LineChart를 모두 지원
- Flutter 전용 설계로 Material 3 테마와 자연스럽게 통합
- pub.dev 커뮤니티에서 가장 활발하게 유지보수되는 차트 라이브러리

## 결과

긍정:
- 수익률 막대 차트(WBS 4.1), 손익 추이 라인 차트(WBS 4.2) 구현 완료
- 터치 툴팁, 그리드, 커스텀 색상 지원

부정 / 제약:
- 실시간 애니메이션이 있는 복잡한 차트가 필요할 경우 `syncfusion`으로 전환 검토

## 후속 작업

- [x] `fl_chart: ^0.70.2` pubspec.yaml에 추가
- [x] WBS 4.1 수익률 막대 차트 구현
- [x] WBS 4.2 누적 손익 라인 차트 구현
