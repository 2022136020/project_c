# Testing

## 수동 테스트 체크리스트

### 인증
- [ ] 회원가입 → 이메일/비밀번호로 계정 생성 성공
- [ ] 로그인 → HomeScreen 진입 확인
- [ ] 앱 재실행 → 자동 로그인 확인 (로그인 화면 거치지 않음)
- [ ] 로그아웃 → LoginScreen으로 이동 확인
- [ ] 잘못된 비밀번호 → 에러 메시지 표시 확인

### 계좌 관리
- [ ] 주식 계좌 등록 (이름 + 초기 잔액)
- [ ] 코인 계좌 등록
- [ ] 입금 기록 → 잔액 증가 확인
- [ ] 출금 기록 → 잔액 감소 확인
- [ ] 계좌 삭제 → 목록에서 제거 확인

### 포지션 관리
- [ ] 포지션 등록 (종목명, 진입가, 수량, 매수 근거, 목표가, 손절가)
- [ ] 보유 중 목록에 표시 확인
- [ ] 포지션 정리 → 수익률·손익 계산 확인
- [ ] 정리 후 복기 메모 작성 → 저장 확인
- [ ] 정리 완료 목록으로 이동 확인

### 대시보드
- [ ] 총 잔액 합산 표시 확인
- [ ] 누적 실현 손익 계산 확인
- [ ] 승률 계산 확인
- [ ] 최근 포지션 5개 표시 확인

### 차트
- [ ] 수익률 막대 차트 표시 (정리 포지션 있을 때)
- [ ] 누적 손익 라인 차트 표시
- [ ] 기간 필터 (1주/1달/3달/6달/1년/전체) 동작 확인
- [ ] 포지션 없을 때 빈 화면 메시지 확인

## 빌드 검증

### 웹 빌드
```bash
flutter build web
# → build/web/ 생성 확인
```

### Android 빌드
```bash
flutter build apk --release
# → build/app/outputs/flutter-apk/app-release.apk 생성 확인
```

### 정적 분석
```bash
flutter analyze
# → No issues found
```

## Firebase 보안 규칙 검증

아래 규칙이 적용되어 있는지 확인:
```
사용자 A가 사용자 B의 데이터를 읽을 수 없어야 함
로그인하지 않은 상태에서 데이터 접근 불가
```

Firebase Console → Firestore → 규칙 탭에서 확인.
